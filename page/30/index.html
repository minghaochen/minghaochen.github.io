美音发音技巧——末尾s的发音

2019-08-23
English

不知道你会不会傻傻分不清楚

likes /s/ 还是 /z/？

lives /s/ 还是 /z/？

首先说说 /s/ 和 /z/ 的发音，需要注意的是这个 /z/ 的发音其实和 /s/ 是一样的！只是在 /s/ 的基础上加上了声带的震动， 不是中文拼音里的 z，是中文拼音里的 z，是中文拼音里的 z！！！ 重要的事情说三遍，很多小伙伴很可能都是按照拼音的 ‘zi’ 来发音(包括我之前也是)那就要好好改改咯。

知道了怎么发音我们再来看几个单词，你试着读读看？

Set 1: likes/books/laughs/mats

Set 2: lives/feels/cares

Set 3: is/has/was/these/because

答案揭晓：第一组都发 /s/，第二组都发 /z/，第三组都发 /z/

不知道你发对了没有还是 /s/ 和 /z/ 胡乱发一通，搞不清楚也没关系，我们先来找找规律，第一组 s 前都是什么？对！是以清辅音 结尾的；第二组呢？是以浊辅音结尾的，第三组则是元音结尾的。

那你就记住：/s/ 是清辅音，那就接在清辅音后面，/z/ 是浊辅音那就接在浊辅音后面，元音后面通常跟 /z/，OK，再试试把上面三组单词读一遍感受一下。

问题又来了，有的小伙伴会问什么是清辅音，什么是浊辅音，这个很好判断，你摸着你的声带，发音时感受到声带的就是浊辅音，比如/b/,/d/,/g/,/v/,/z/，不震动的就是清辅音，比如/p/,/t/,/k/,/f/,/s/。

还有一个关于浊辅音和清辅音的技巧就是：浊辅音前的元音发音更“长”一点

不信你对比一下以下几组单词： back 和 bag; safe 和save; pick 和 pig; lock 和 log; cup 和 cub

是不是发音每组单词中后面那个单词的元音都拉得比较长呢，那就对了！
